### 3. Scrolling Performance Fix ⭐⭐⭐ CRITICAL

**Problem:** Lag on complex sites (Twitter, Reddit, Facebook)

**Root Causes:**
- Hardware acceleration not enabled
- Excessive repaints
- JavaScript blocking main thread
- Extension overhead

**Planned Fixes:**

**Hardware Acceleration:**
```cpp
// Enable GPU acceleration
webkit_settings_set_hardware_acceleration_policy(settings, 
    WEBKIT_HARDWARE_ACCELERATION_POLICY_ALWAYS);

// Enable compositing
webkit_settings_set_enable_accelerated_2d_canvas(settings, TRUE);
```

**Rendering Optimization:**
- Enable WebGL
- Use offscreen rendering for heavy pages
- Throttle animations when tab not visible
- Reduce extension content script frequency

**Settings Toggle:**
- Hardware acceleration on/off
- Smooth scrolling quality (low/medium/high)
- Animation effects toggle
- Force GPU compositing

**Benchmarks:**
- Test on Twitter, Reddit, YouTube
- Measure: FPS, frame drops, scroll smoothness
- Target: 60 FPS on complex pages

### 5. Bookmarks Import/Export Enhancement ⭐⭐ HIGH PRIORITY

**Problem:** Current implementation is basic, needs polish

**Improvements:**

**Import:**
- **Better file format support:**
  - Chrome HTML bookmarks
  - Firefox JSON bookmarks
  - Safari Property List
  - Edge HTML bookmarks
  - Opera bookmarks
  - Pocket export
- **Import options:**
  - Merge with existing bookmarks
  - Replace all bookmarks
  - Import to specific folder
  - Import only bookmarks (skip folders)
  - Import only folders
- **Duplicate handling:**
  - Skip duplicates
  - Keep both
  - Replace existing
- **Progress indicator:**
  - Show import progress
  - Success/failure count
  - Preview before import
- **Validation:**
  - Check URL validity
  - Warn about broken bookmarks
  - Fix common issues automatically

**Export:**
- **Format options:**
  - HTML (Netscape format)
  - JSON (Braya format)
  - CSV (for Excel)
  - Markdown (for documentation)
- **Export filters:**
  - Export all
  - Export folder
  - Export selected
  - Export by date range
- **Options:**
  - Include folders/hierarchy
  - Include tags
  - Include timestamps
  - Include favicons
- **Scheduled exports:**
  - Auto-backup bookmarks daily/weekly
  - Save to cloud (Dropbox/Google Drive)

**UI Flow:**
```
Bookmarks Manager → Import/Export
├── Import
│   ├── Choose file → Select format (auto-detect)
│   ├── Preview (show what will be imported)
│   ├── Options (merge/replace, duplicates)
│   └── Import → Success message with stats
└── Export
    ├── Choose what to export
    ├── Select format
    ├── Options (include folders, dates, etc.)
    └── Save file → Success message
```

**Technical Implementation:**
```cpp
class BookmarkImporter {
    enum class Format {
        AUTO_DETECT,
        CHROME_HTML,
        FIREFOX_JSON,
        SAFARI_PLIST,
        BRAYA_JSON,
        CSV,
        MARKDOWN
    };
    
    struct ImportOptions {
        Format format;
        bool merge;
        std::string targetFolder;
        DuplicatePolicy duplicatePolicy;
    };
    
    std::vector<Bookmark> parseHTML(const std::string& file);
    std::vector<Bookmark> parseJSON(const std::string& file);
    std::vector<Bookmark> parsePlist(const std::string& file);
    Format detectFormat(const std::string& file);
    void validateBookmarks(std::vector<Bookmark>& bookmarks);
};

class BookmarkExporter {
    void exportToHTML(const std::vector<Bookmark>& bookmarks);
    void exportToJSON(const std::vector<Bookmark>& bookmarks);
    void exportToCSV(const std::vector<Bookmark>& bookmarks);
    void exportToMarkdown(const std::vector<Bookmark>& bookmarks);
};
```
