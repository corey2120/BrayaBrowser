# Braya Browser - Version 1.0.5 Development Plan

**Created**: November 13, 2025
**Released**: November 13, 2025
**Status**: Completed - Bookmark Folders Implemented

## Overview
Version 1.0.5 focuses on UX refinements, Vivaldi-style settings organization, and improved bookmark management to match the experience of modern browsers like Firefox, Chrome, and Zen.

---

## Priority 1: Bookmark Folders (Like Zen/Firefox/Chrome)

### Current State
- Bookmarks bar shows flat list of bookmarks
- No folder organization
- No nested bookmark structure

### Goal
Implement Chrome/Firefox/Zen-style bookmark folders with:
- Dropdown menus for folders
- Nested folder support
- Folder icons in bookmarks bar
- Right-click context menu on folders
- Drag-and-drop organization

### Implementation Tasks
- [x] **Data Structure** - Add folder support to bookmark storage ✅
  - ✅ Extended `BrayaBookmarks::Bookmark` struct with `isFolder` and `children` fields
  - ✅ Updated JSON serialization/deserialization
  - ✅ Backward compatibility with existing bookmarks maintained

- [x] **UI Components** - Folder display in bookmarks bar ✅
  - ✅ Created folder button with icon (📁)
  - ✅ Implemented popup menu on click
  - ✅ Show nested bookmarks in dropdown
  - ✅ Recursive folder rendering implemented
  - ✅ Folder hover effects working

- [x] **Dialog Enhancements** - Add/Edit bookmark with folder selection ✅
  - ✅ Folder dropdown in add bookmark dialog
  - ✅ "Create New Folder" option available
  - ✅ Move bookmark between folders via edit dialog

- [x] **Context Menu** - Right-click options ✅
  - ✅ Delete folder (with confirmation)
  - ✅ Right-click on bookmarks inside folders
  - ✅ "Move to Bookmarks Bar" from folder (via context menu)
  - ✅ Delete bookmark from folder (via context menu)
  - ✅ UI refresh callback system implemented
  - ⚠️ "Open All in Tabs" - Deferred to future version
  - ⚠️ Edit folder name - Deferred to future version

- [x] **Drag & Drop** - Reorganize bookmarks ✅
  - ✅ Drag bookmark into folder (works perfectly)
  - ✅ Reorder bookmarks on bookmarks bar
  - ✅ Move folders around on bookmarks bar
  - ✅ Visual feedback during drag operations
  - ⚠️ **Drag OUT of folders** - GTK4 popover limitation prevents drag initiation
    - **Workaround Implemented**: Right-click → "📤 Move to Bookmarks Bar"
    - This is a GTK4 constraint: popovers grab input events, preventing drag operations
    - Context menu approach is standard solution used by GTK applications
  - ⚠️ Reorder bookmarks within folder - Deferred to future version

### Files Modified ✅
- ✅ `src/BrayaBookmarks.h` - Added folder data structures, refresh callback support
- ✅ `src/BrayaBookmarks.cpp` - Implemented folder logic, UI rendering, and context menus
- ✅ `src/BrayaWindow.cpp` - Updated bookmark bar rendering with refresh callback
- ✅ `CMakeLists.txt` - Updated version to 1.0.5

### References
- Firefox Bookmarks Toolbar: Folders with dropdown menus
- Chrome Bookmarks Bar: Clean folder icons with popups
- Zen Browser: Modern folder styling

## Risk Assessment

### High Risk
- Settings redesign might break existing preferences
  - **Mitigation**: Careful migration code, extensive testing

### Medium Risk
- Bookmark folder implementation could corrupt existing bookmarks
  - **Mitigation**: Auto-backup before migration, validation checks

### Low Risk
- Toolbar customization might conflict with existing UI
  - **Mitigation**: Default to current layout, graceful fallbacks
