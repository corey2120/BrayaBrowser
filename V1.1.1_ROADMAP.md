### 🟠 P1 — Tor & Onion Support

The headline feature for v1.1.1. Route traffic through Tor so `.onion` sites work
natively and regular browsing can optionally go through the Tor network.

**How it works:**
1. **Tor daemon** — spawn/manage a `tor` process (or connect to a running one on `127.0.0.1:9050`)
2. **SOCKS5 proxy** — configure WebKit's network session to route through Tor's SOCKS5 port
3. **Private Tor window** — new window type (extends private/incognito) that forces all traffic through Tor
4. **`.onion` detection** — intercept navigation to `.onion` URLs and auto-enable Tor mode for that tab
5. **Tor status indicator** — toolbar badge (onion icon) showing Tor connected/connecting/off

**Implementation plan:**
- `BrayaTorManager` class — starts/stops tor process, monitors control port, exposes status
- `WebKitNetworkSession` proxy config via `webkit_network_session_set_proxy_settings`
- New "Open Private Tor Window" menu item (Ctrl+Shift+T shortcut)
- Onion icon resource added to `resources/`
- Settings toggle: "Route all traffic through Tor" (off by default)
- Dependency: `tor` package (optional — graceful degradation if not installed)

**Security notes:**
- Tor window disables WebRTC (IP leak prevention)
- No mixed-mode: a Tor tab stays on Tor for its lifetime
- JavaScript stays enabled by default (matches Tor Browser behavior for usability; user can disable per-site)

### 🟡 P2 — Small Quality-of-Life Updates

#### URL Bar improvements
- Show full URL on focus, display readable title/domain when unfocused
- Paste-and-go: middle-click on URL bar pastes clipboard and navigates immediately

#### Tab improvements  
- Tab search: Ctrl+Shift+A opens a quick tab switcher (list of open tabs, fuzzy search)
- Drag-to-reorder tabs in the sidebar
- Tab count badge on the window title when >10 tabs open

#### Downloads
- Download progress in the taskbar / dock badge (via GApplication progress)
- "Open containing folder" button on completed downloads

#### Bookmarks
- Quick-add shortcut: Ctrl+D adds current page instantly (no dialog, toast confirmation)
- Bookmark folders in the bookmarks bar

#### Reader Mode
- Estimated read time displayed in reader mode toolbar
- Font size controls in reader mode (±)

#### History
- Search/filter in history panel
- "Clear history from this site" right-click option

#### Keyboard shortcuts
- Ctrl+Shift+U — view page source
- Ctrl+= / Ctrl+- — zoom in/out (currently only in per-site settings)
- F6 — focus URL bar (standard browser shortcut)

#### Misc
- `about:braya` home page: add quick-access tiles for most-visited sites
- Status bar: show hovered link URL (already planned, make sure it works)
- Remember window size/position between sessions
